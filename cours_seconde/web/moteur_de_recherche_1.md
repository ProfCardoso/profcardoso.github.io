# Comment fonctionne un moteur de recherche ?

## Vocabulaire

En informatique, une **requête** est une interrogation d’une base de données. Elle peut comporter un certain nombre de critères pour préciser la demande.

##  Présentation de l'activité

Vous disposez de six textes.  
L’objectif est de comprendre comment un moteur de recherche classerait ces textes en fonction des mots-clés utilisés pour la recherche.

2) Proposer 1 ou 2 critères que l'on pourrait ajouter.
