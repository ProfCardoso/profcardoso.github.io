# TP - Parcours Algoréa

Pour continuer le travail de découverte des données et de leur traitement, nous allons utiliser le site de <a href="https://parcours.algorea.org/fr/a/snt-donnees;a=0" target="_blank">Parcours Algorea</a>.

## Inscription

Avant de commencer les différentes activités, il faudra d'abord vous <a href="https://login.france-ioi.org/register" target="_blank">créer un compte</a> sur la plateforme. 

Attention, votre identifiant est unique, et pour faciliter le suivi des activités, il devra respecter les règles suivantes :
- Contenir `2gt` + le numéro de votre classe + `-` + la première lettre de votre nom + la première lettre de votre prénom

Votre professeur vous donnera ensuite le code associé pour rejoindre votre groupe classe. 

## Activité Découverte

Après la création de votre compte, faites le maximum d'exercices pour vous approprier les notions de données structurées et de leur traitement !
