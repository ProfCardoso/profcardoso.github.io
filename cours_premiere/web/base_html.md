# HTML et CSS

## Application 

Pour la suite du cours sur HTML et CSS, nous allons devoir utiliser une application pour écrire et voir le résultat en direct des pages webs.

Cette application, c'est Visual Studio, avec quelques extensions à ajouter :

- **HTML Preview**
- **HTML CSS Support**
- et d'autre encore si vous souhaitez des améliorations ou d'autre chose encore sur votre application.

## Pour les plus rapides 

Vous pouvez, dans votre dossier NSI, créer un dossier `page_web`, où se situera votre site web, puis à <a href="https://developer.mozilla.org/fr/docs/Learn_web_development/Getting_started/Your_first_website"> cette adresse </a>, commencer à suivre les étapes pour débuter et créer votre site web.
